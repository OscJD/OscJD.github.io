2021 Design and Development of an Automatic Mosquitos Catch-counter and Classifier Device powered by DeepLearning.

This project aims to recognize and count pests, with the intention of identifying their behavior in order to generate current information and predictions that can help in decision-making. outdoor This was done with a standalone outdor device, it is made up of a 10 MP camera, a solar energy module and communicates through a cellular network This hardware is configured to take daily photographs These images are also integrated into an application that facilitates the classification of the insects found in the traps by experts in the area and that at the same time works to display a summary of the data by date. Using image processing techniques, the area corresponding to the adhesive trap is identified. The identified image is processed by an Artificial Neural Network that is responsible for determining the type of insect.

2020 Real time Trays detection and count for Industrial Monitoring

This project focused on the counting and recognition of assets, in this case a plastic tray where the products are transported. For this challenge, a method called Fast Region-based Convolutional Network (Fast R-CNN) was applied, this method takes as input a complete image and a set of object proposals. Another challenge of this project was the use of wide-angle lenses, due to the limited space that exists between the camera and the entrance to the pit, wide-angle lenses had to be adapted to cover the entire panorama relevant to the recognition. The use of these lenses generates a radial distortion in the image. To correct this distortion, a calibration was carried out on the equipment, finding the intrinsic parameters.
